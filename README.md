  - add_index : インデックスを貼る

## Association
### Usres 
- has_many :Messages
- has_many :Groups, through: :Users_Groups
- has_many :Users_Groups


### Groups
- has_many :Messages
- has_many :Users, through: :Users_Groups
- has_many :Users_Groups

### Users_Groups
- belongs_to :User
- belongs_to :Group

### Messages
- belongs_to :User
- belongs_to :Group
